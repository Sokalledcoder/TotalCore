# https://tradingview.github.io/lightweight-charts/docs/api/interfaces/BarStyleOptions

# Interface: BarStyleOptions

Represents style options for a bar series.

## Properties

### upColor

upColor:`string`

Color of rising bars.

#### Default Value

`'#26a69a'`

### downColor

downColor:`string`

Color of falling bars.

#### Default Value

`'#ef5350'`

### openVisible

openVisible:`boolean`

Show open lines on bars.

#### Default Value

`true`

### thinBars

thinBars:`boolean`

Show bars as sticks.

#### Default Value

`true`
