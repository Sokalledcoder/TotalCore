# Enumeration: LineStyle

Represents the possible line styles.

## Enumeration Members

### Solid

Solid:`0`

A solid line.

### Dotted

Dotted:`1`

A dotted line.

### Dashed

Dashed:`2`

A dashed line.

### LargeDashed

LargeDashed:`3`

A dashed line with bigger dashes.

### SparseDotted

SparseDotted:`4`

A dotted line with more space between dots.
