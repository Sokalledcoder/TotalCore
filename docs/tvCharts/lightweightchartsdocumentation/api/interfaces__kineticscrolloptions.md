# Interface: KineticScrollOptions

Represents options for enabling or disabling kinetic scrolling with mouse and touch gestures.

## Properties

### touch

touch:`boolean`

Enable kinetic scroll with touch gestures.

#### Default Value

`true`

### mouse

mouse:`boolean`

Enable kinetic scroll with the mouse.

#### Default Value

`false`
